# New Page SEO Checklist
**Project**: apexengineering.org.in

---

## 📝 Rules for Adding New Products or Services

If you add a new page to the website in the future, follow this checklist to preserve search visibility:

### 1. Title and Metadata
* [ ] **Page Title**: Length 50–60 characters. Format: `[Primary Keyword] — [Value Proposition] | Apex Engineering` (e.g. `Aluminium Workbenches — Custom Modular Frames | Apex Engineering`).
* [ ] **Meta Description**: Length 130–150 characters. Must describe the page concisely and include a call to action.
* [ ] **Canonical URL**: Add a self-referencing canonical tag with a trailing slash (`https://apexengineering.org.in/new-page/`).

### 2. Header and Semantic Structure
* [ ] **Single H1 Tag**: Exactly one `<h1>` tag at the top containing the primary keyword.
* [ ] **Heading Order**: Nested logically without skipping levels (H1 $\rightarrow$ H2 $\rightarrow$ H3).
* [ ] **Semantic Elements**: Wrap page parts in semantic elements (`<main>`, `<section>`, `<footer>`).

### 3. Media & Optimization
* [ ] **WebP Format**: Convert all new catalog photos to compressed `.webp` format.
* [ ] **ALT Text**: Provide descriptive, keyword-rich ALT tags (e.g. `alt="custom modular aluminium workbench assembly"` instead of `alt="img1"`).
* [ ] **Explicit Sizes**: Set width and height properties on images to prevent layout shifting.

### 4. Crawler Submission
* [ ] **Sitemap Update**: Ensure the new URL is included in your static or dynamic `sitemap.xml`.
* [ ] **Google Search Console**: Input the new URL inside GSC's URL Inspection tool and click "Request Indexing".
